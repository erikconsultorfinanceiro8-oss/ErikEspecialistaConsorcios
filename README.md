# Erik Simon - Especialista em Consórcios

Site profissional de apresentação e calculadora de consórcios.

## Como usar

1. Faça upload desta pasta para um repositório GitHub
2. Vá em Settings → Pages
3. Selecione "Deploy from a branch"
4. Escolha a branch "main"
5. Seu site estará online em: `https://seu-usuario.github.io/seu-repositorio`

## Domínio personalizado

Para usar um domínio personalizado (ex: eriksimon.com.br):
1. Configure os registros DNS do seu domínio
2. Adicione o domínio em Settings → Pages → Custom domain
3. Aguarde a propagação DNS
